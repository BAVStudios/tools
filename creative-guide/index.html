What is XcYv?

XcYv is how we describe how many creatives to produce. It stands for X concepts, Y variations.

  • Concept = a unique creative idea, angle, or message. Each concept gets its own series letter (A, B, C...)
  • Variation = different versions of the same concept. Same idea, different execution (different headline, different image, different layout). Each variation gets a number (01, 02, 03...)
Example: 3c3v

3 concepts x 3 variations = 9 unique creatives

Concept A: 3 variations = A_01, A_02, A_03 Concept B: 3 variations = B_01, B_02, B_03 Concept C: 3 variations = C_01, C_02, C_03 Total: 9 creatives

Formats: Feed and Story

Most creatives need to be produced in two formats:

FormatSuffixAspect ratioWhere it shows
Feed-F1:1 or 4:5Facebook/Instagram feed, right column
Story-S9:16 (vertical)Stories, Reels, full-screen placements
This doubles the file count

3c3v = 9 creatives. Each in Feed AND Story = 18 files total.

Always check: if there's a -F version, there should be a -S version too.

How XcYv Maps to Meta Ads Manager

Each concept becomes its own ad set. Each variation becomes an ad inside that ad set.

Courtesy Creatives

Sometimes we produce extra creatives outside the regular monthly quota. This happens when:

  • Underperformance - ads aren't hitting targets, we need fresh creative to test
  • Promotions - a client runs a sale, launch, or seasonal event that needs dedicated creatives
  • Opportunities - something timely (trending moment, competitor gap) worth jumping on

These are produced as a courtesy on top of the regular quota. They follow the same naming convention and upload process.

Creative File Name Format

Every creative file from Raka follows this pattern:

{Y}{MM}{C}_{VARIATION}_{FORMAT}_{TITLE}-{RATIO}.ext
PartWhat it meansExamples
YLast digit of the year6 = 2026, 5 = 2025
MMMonth (01-12)03 = March, 11 = November
CConcept letterA = concept 1, B = concept 2, C = concept 3
VARIATIONVariation number within that concept01, 02, 03
FORMATContent typeIMG (static), UGC, EGC, VID, EDM
TITLEDescriptive name of the conceptSkincare Undone, Breakout Tax
-RATIOAspect ratio (no space before hyphen)-S = story (9:16), -F = feed (1:1/4:5)

Reading a File Name

603A_01_IMG_Skincare Undone-S.png 603 = 2026, March A = concept A (first concept) 01 = variation 1 IMG = static image Skincare Undone = concept name -S = story format (9:16)

The YMMC part (603A) tells you which ad set this file belongs to in Meta Ads Manager.

Where Files Live

Creative files are stored in Google Drive:

10 BAVS Clients > [Client Name] > 01 Ads/
  • Files are organized by client
  • Every file should be identifiable on its own - you shouldn't need the folder path to understand what it is
  • If a -S version exists, there should be a -F version too
Quick decode
  • First 3 characters = when (year + month)
  • Letter after that = which concept
  • Number after underscore = which variation
  • Last letter before .ext = which format (-S or -F)

Your Job: Make Sure We Hit These

  • Track progress in Asana - creative tasks are assigned to Raka. Check if they're on track early in the month, not at the end.
  • Flag shortfalls early - if it's the 15th and a client is behind on their quota, raise it in #bav-training
  • Count correctly - 3c3v = 9 unique creatives. Each in F + S = 18 files. Don't count file count as creative count.
  • Promos are extra - promo creatives don't count toward the monthly quota. They're on top of it.
  • Courtesy creatives are extra too - if we produce extra due to underperformance, that doesn't reduce next month's quota
Monthly checklist
  • Week 1: Check Asana for creative tasks assigned this month. All clients covered?
  • Week 2: Check progress. Are designs started? Flag any blockers.
  • Week 3: Creatives should be arriving. Upload as they come in.
  • Week 4: Final check. Did every client get their quota? Log any gaps.
