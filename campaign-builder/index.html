CBO vs ABO - When to use which

TypeWhenWhy
ABOCT AT NA Testing (most of the time)You control the budget for each ad set separately. Every test gets a fair chance.
CBOSC Scaling winnersMeta automatically sends more budget to the best-performing ad sets. Use this when you already know what works.
CBOLarge budget testing (rare)Sometimes with big budgets, CBO works for testing too. Juan will tell you when.

Not sure? Use ABO. You can always move winners into a CBO SCALE campaign later.

How to Find the Right Campaign

Before creating a new campaign, check if the right one already exists. Look for these three things in order:

1
Objective - CV for sales (most common), LD for leads, AW for awareness
2
Agenda - CREATIVE TEST, SCALE, AUDIENCE TEST, etc.
3
Highest series number - that's the most recent campaign
Example
You have new creatives for a sales campaign. Which campaign do you use?
A. [BAVS] CV || 03 || PUR || ABO || CREATIVE TEST B. [BAVS] CV || 07 || PUR || ABO || CREATIVE TEST ← this one C. [BAVS] CV || 09 || PUR || CBO || SCALE D. [BAVS] LD || 12 || WEB || CBO || CREATIVE TEST
B - It's CV (sales) + CREATIVE TEST + highest series number (07). C is wrong because it's SCALE. D is wrong because it's LD (lead gen). A is older (series 03).
Can't find a match?
Create a new campaign. Use the next series number after the highest one that exists.
Don't
  • Put new creatives in a SCALE campaign
  • Use CBO for testing (unless Juan says to)
  • Reuse a series number - always go up by 1
  • Pick a campaign just because one word matches - check objective AND agenda AND series number

Series - The Most Important Field

Series tells you what this ad set is for and which ads belong in it.

CT YMMC - Creative Testing

Y = last digit of year, MM = month, C = concept letter

6 = 2026 03 = March A = concept A 603A = 2026 March concept A 507H = 2025 July concept H

A YMMC ad set must only contain ads from that exact batch. 603A ad set = 603A_01, 603A_02, 603A_03.

SC AT NA AUD - Audience / Scaling

Just the letters AUD. Used for scaling winners or testing audiences. You can put any proven ads in here - best performers from any month, any concept.

Always Create New Ad Sets

Never add new ads to an existing ad set. Every new batch of creatives or new audience test = a fresh new ad set. This keeps the data clean and avoids resetting the learning phase.

Don't
  • Add new creatives to an existing ad set
  • Increase budget on a winning test ad set (this resets learning - create a new one in SCALE instead)
  • Use anything other than BROAD targeting for creative tests
  • Use creative types outside of IMG / UGC / EGC / VID / CRL / MIX

Understanding Creative File Names

When Raka delivers creative files to Google Drive, the file names follow this pattern:

{Y}{MM}{C}_{VARIATION}_{FORMAT}_{TITLE}-{RATIO}.ext
603A_01_IMG_Skincare Undone-S.png 6 = 2026 03 = March A = concept A 01 = variation 1 IMG = static image -S = story format (9:16) -F = feed format (1:1 or 4:5)

The YMMC part of the filename matches the ad set series. A file called 603A_01 goes in the ad set named 603A.

Post ID - Reusing Existing Ads

When scaling or audience testing, you don't upload the creative again. Instead, you reuse the existing ad through its post ID. This keeps all the likes, comments, and shares (social proof) from the original ad.

How to get a post ID:

  1. Go to the winning ad in Ads Manager
  2. Click the preview or "See Post with Comments"
  3. Copy the ID from the URL
  4. When creating a new ad, select "Use existing post" and paste the ID
